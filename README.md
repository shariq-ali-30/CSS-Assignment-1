# Periodic Table Design

## About The Project

This project is a periodic table design created using HTML and CSS.

The purpose of this assignment was to practice CSS styling, colors, layouts, hover effects, and organizing elements using classes while creating a visual representation of chemical elements.

## Technologies Used

- HTML5
- CSS3

## Features

- Complete periodic table layout
- Different colors for element categories
- Styled element boxes
- Hover animation effects
- Atomic number, symbol, and element name display
- Responsive structured layout using tables
- Custom styling with CSS classes

## What I Learned

Through this assignment, I learned:

- How to connect CSS with HTML
- How to use CSS classes for styling
- How to apply colors and spacing
- How to create hover effects using pseudo classes
- How to style text and elements
- How transitions and transforms work in CSS
